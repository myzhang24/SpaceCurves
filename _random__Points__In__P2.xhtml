SpaceCurves :: randomPointsInP2

randomPointsInP2 -- Creates a list of ideals of random points in P^2

Description

This method produces a list of ideals of random points (over the base field) in P^2.

Synopsis

  • Usage:
    L = randomPointsInP2(n,R)
  • Inputs:
    • n, an integer, number of random points
    • R, a ring, polynomial ring of P^2
  • Outputs:
    • L, a list, of ideals of points
If the Ring input is omitted, the default ring is ZZ/101[x_0..x_2].
