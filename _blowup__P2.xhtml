SpaceCurves :: blowupP2

blowupP2 -- Blowup P^2 at given points and polarize by a given line bundle

Description

This method produces a PolarizedSurface corresponding to the blowup of P^2 at a list of points, and polarized by a given divisor.

A basis of the numerical group is given by pi^*O(1), -E_1, ..., -E_n, where pi^*O(1) is the pullback of O(1) on P^2 and E_i are the exceptional divisors of the blowup.

Synopsis

  • Usage:
    X = blowupP2(L,H)
  • Inputs:
    • L, a list, of Ideals of points in P^2
    • H, a list, giving coordinates of the polarizing bundle
  • Outputs:
    • X, an object of class PolarizedSurface
If the polarizing line bundle is omitted from the second input, then the default polarizing bundle is the anti-canonical bundle.

Synopsis

  • Usage:
    X = blowupP2(L)
  • Inputs:
    • L, a list, of Ideals of points in P^2
  • Outputs:
    • X, an object of class PolarizedSurface

If the first input is an integer n, then the method blows up P^2 at n random points.

Synopsis

  • Usage:
    X = blowupP2(n,H)
  • Inputs:
    • n, an integer, number of random points to blowup
    • H, a list, coordinate of the polarizing bundle
  • Outputs:
    • X, an object of class PolarizedSurface

Synopsis

  • Usage:
    X = blowupP2 n
  • Inputs:
    • n, an integer, number of random points to blowup
  • Outputs:
    • X, an object of class PolarizedSurface
i4 : X = blowupP2(1,{2,1})

o4 = P^2 blown-up at 1 points
          polarized by {2, 1}

o4 : PolarizedSurface
i5 : Y = blowupP2 6

o5 = P^2 blown-up at 6 points
          polarized by {3, 1, 1, 1, 1, 1, 1}

o5 : PolarizedSurface
