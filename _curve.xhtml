SpaceCurves :: curve

curve -- Computes the ideal of a random effective divisor

Description

The method curve(Divisor) takes a random effective divisor in the specified class, and computes the ideal of the image under the rational map given by the LinearSystem of the PolarizedSurface it lies on.

Synopsis

  • Usage:
    I = curve D
  • Inputs:
    • D, an object of class Divisor
  • Outputs:
    • I, an ideal, of the image curve

The method curve(ZZ,Divisor) takes a random effective divisor in the specified class, and computes the ideal of the general projection into P^n of the original image under the rational map given by the LinearSystem of the PolarizedSurface it lies on.

Synopsis

  • Usage:
    I = curve(n,D)
  • Inputs:
    • n, an integer, dimension of the general linear subsystem
    • D, an object of class Divisor
  • Outputs:
    • I, an ideal, of the image curve by a general n dimensional linear subsystem
