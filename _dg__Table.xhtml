SpaceCurves :: dgTable

dgTable -- prints the table of (degree,genus) pairs

Description

dgTable prints the table of (degree,genus) pairs, where the horizontal axis is the degree and the vertical is the genus. The input can be a List of Divisors as well as Ideals.

Ways to use dgTable :

  • dgTable(List)
