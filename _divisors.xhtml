SpaceCurves :: divisors

divisors -- Produces a list of Divisors

Description

The method divisors(ZZ,PolarizedSurface) produces a List of all Divisors of a given degree on a given PolarizedSurface.

Synopsis

  • Usage:
    L = divisors(d,X)
  • Inputs:
    • d, an integer, degree of the divisors
    • X, an object of class PolarizedSurface
  • Outputs:
    • L, a list, of all divisors of degree d on X
i1 : X = scroll(1,2)

o1 = A scroll of type {1, 2}

o1 : PolarizedSurface
i2 : divisors(6,X)

o2 = {{3, 3}, {2, 4}, {1, 5}}

o2 : List

The method divisors(ZZ,ZZ) produces a non-exhaustive List of Divisors of a given degree and genus across different PolarizedSurfaces.

Synopsis

  • Usage:
    L = divisors(d,g)
  • Inputs:
    • d, an integer, degree of the Divisors
    • g, an integer, genus of the Divisors
  • Outputs:
    • L, a list, of Divisors
i3 : divisors(5,8)

o3 = {}

o3 : List
