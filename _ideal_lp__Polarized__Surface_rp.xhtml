SpaceCurves :: ideal(PolarizedSurface)

ideal(PolarizedSurface) -- Computes the ideal of the image surface

Synopsis

  • Function: ideal

Description

Given a PolarizedSurface X, the LinearSystem specifies a map from the polynomial ring kk[x_0..x_n] to R, the section ring of the hyperplane bundle H. The kernel of this map kk[x_0..x_n] -> R is the ideal of the image of the corresponding rational map X --> P^n.

Synopsis

  • Usage:
    I = ideal X
  • Inputs:
    • X, an object of class PolarizedSurface
  • Outputs:
    • I, an ideal
