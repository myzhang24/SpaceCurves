SpaceCurves :: PolarizedSurface

PolarizedSurface -- A surface with a linear system

Description

PolarizedSurface is a type of HashTable storing the information of a surface with a linear system.

The key SurfaceData stores a List encoding defining characteristics of the PolarizedSurface. If the surface is the blowup of P^2 at finitely many points, then this list stores the list of ideals of these points. If the surface is the scroll, then the list stores the type {a,b}.

The key IntersectionPairing stores a Matrix encoding the intersection matrix of the numerical group of the surface. A basis of the numerical group is implicitly chosen.

The key HyperplaneClass contains a List encoding the coordinates of the polarizing bundle with respect to the same basis as in the IntersectionPairing.

The key CanonicalClass contains a List encoding the coordinates of the canonical bundle with respect to the same basis as in the IntersectionPairing.

The key LinearSystem stores a row Matrix whose entries form a vector space basis of the linear system of the polarizing bundle.

The key Chi stores the Euler characteristic of the sheaf O_X.

i1 : X = blowupP2(1,{2,1})

o1 = P^2 blown-up at 1 points
          polarized by {2, 1}

o1 : PolarizedSurface
i2 : peek X

o2 = PolarizedSurface{CanonicalClass => {-3, -1}                                                                     }
                      Chi => 1
                      HyperplaneClass => {2, 1}
                      IntersectionPairing => | 1 0  |
                                             | 0 -1 |
                      LinearSystem => | x_1^2+19x_1x_2 x_1x_2+19x_2^2 x_0^2+21x_0x_2 x_0x_1+21x_1x_2 x_0x_2+21x_2^2 |
                      SurfaceData => {ideal (- 36x  - 24x , - 30x  - 24x , - 30x  + 36x )}
                                                  0      1       0      2       1      2

In the above example, the surface in question is the blowup of P^2 at a point p. A basis of its numberical group is given by pi^*O(1) and E, where pi^*O(1) is the pullback of the hyperplane class of P2 and E is the exceptional divisor of the blowup. The second input is {2,1}, which is the coordinates of the polarizing bundle. Thus we are mapping the blowup of P^2 at p by the linear system of conics passing through p.
