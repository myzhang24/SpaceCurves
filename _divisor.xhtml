SpaceCurves :: divisor

divisor -- Creates a Divisor on a PolarizedSurface

Description

Creating a Divisor on a PolarizedSurface.

Synopsis

  • Usage:
    D = divisor(L,X)
  • Inputs:
    • L, a list, of coordinates of the divisor
    • X, an object of class PolarizedSurface
  • Outputs:
    • D, an object of class Divisor
i1 : X = scroll(1,2)

o1 = A scroll of type {1, 2}

o1 : PolarizedSurface
i2 : D = divisor({2,3},X)

o2 = {2, 3}

o2 : Divisor
